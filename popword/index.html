Chrome 扩展

PopWord 弹词

在网页上左键 / 右键切换单词，搭配发音、浮动展示与学习记录，边浏览边记单词。

立即下载 ⭐ 在 GitHub 查看
提示：无需下载，在当前页面随意点击鼠标左、右键即可在线体验！

功能亮点

👆

点击即出词

左键下一个、右键上一个，支持自定义键盘快捷键，任意网页都能用。

🔀

随机顺序

词库加载后自动打乱，每次都是新顺序，避免固定顺序记忆。

🔊

发音与词库

自动读词、多词库切换（如 CET4），可自定义音频 API，预加载减少卡顿。

📋

学习记录

自动记录学习历史，监控面板随时查看，支持一键关闭弹词、重置词库与清理缓存。

安装方法

  1. 下载压缩包到本地并解压
  2. 打开 Chrome，进入 chrome://extensions/
  3. 开启右上角「开发者模式」
  4. 点击「加载已解压的扩展程序」，选择项目目录即可
